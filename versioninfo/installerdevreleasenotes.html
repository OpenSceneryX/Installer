Version 2.2.1 - 12 August 2018

  1. Fixed 'Unhandled Exception' bug at very end of install process on fast machines.

Version 2.2.0 - 8th August 2018

  1. The intensive local file checking process is now threaded, improving application responsiveness and redraw. The improvements are especially noticeable on Windows.
  2. Both intensive file operations now yield some time to other threads to improve application responsiveness and redraw. The improvements are especially noticeable on Windows.
  3. The install operation is now much faster, and can make full use of fast Internet connections.
  4. The Mac and Windows installers are now code signed, meaning they will not be blocked from running.
  5. The Windows installer now runs as an administrator, using the proper UAC mechanism, specified in the application's Manifest.
  6. The Mac installer is now distributed as a Disk Image (DMG) with a custom design.
  7. Cross-platform bugs in the license confirmation dialog fixed.

Version 2.1.0 - 29th July 2018

  1. The Mac installer now provides support for the default Steam location.  A new button allows you to select the Steam folder if it exists.
  2. The installer now includes the Backup Library that chris k originally built and einstein took control of.  This means that if you have OpenSceneryX installed, you no longer need the Backup Library.  The installer allows you to choose whether the placeholders used are visible or invisible – the visible ones should be bright red.
  3. The Mac version is now a full Cocoa application.
  4. There are three Linux versions: x86-32 bit, x86-64 bit and ARM 32 bit.
  5. The installer should be a bit more robust if your network connection is interrupted.
  6. We now support high resolution displays.
  7. Switched to secure HTTPS for all server communications. This required a rebuild of the network code.
  8. Small design refresh.

Version 2.0.2 - 25th March 2012

  1. Fixed a bug where the installer would fail to create files when X-Plane is installed on a second partition or drive.

Version 2.0.1 - 12th February 2012

  1. Fixed a bug in the Linux version that would cause the download to stop after a certain number of files, usually around 500.
  2. Downloader was creating two temporary files for every download, instead of one.

Version 2.0.0 - 5th February 2012

  1. Installer has been completely re-vamped to be more user-friendly and to describe what it is doing at each stage.
  2. About box window now has a title.
  3. Issues with the installer failing to operate on Vista have been resolved.
  4. Installer has been built with the latest version of the development environment and libraries.
  5. New language translations have been added.

Version 1.2.0 - 9th May 2008

  1. First version which checks for a new version of the installer.
  2. Made the logo clickable to visit the OpenSceneryX website.
